Использование генеративного ИИ в проекте «Выставка Диафрагма»

1. Структура и вёрстка страниц авторов (author-1.html — author-4.html)
Цель: создать четыре страницы с одинаковой структурой, но разными изображениями, текстами и цветовыми акцентами. ИИ написал HTML-разметку для каждой страницы: герой-блок с портретом и девайсом, галерею с различными форматами раскладок (три в ряд, дуэт, одиночные кадры, текст рядом с фото), тематическую цветную секцию, футер с именем автора.

2. Главная страница (index.html) — интерактивный фон
Цель: при наведении на имя автора фоновое изображение меняется. ИИ написал JavaScript-логику смены src фонового изображения и показа превью-карточки с портретом автора при событиях mouseenter / mouseleave.

3. Навигация — логотип «Диафрагма» с декоративными блоками
Цель: добавить на все страницы фирменный элемент с эффектом наложения. ИИ написал CSS с псевдоэлементами ::before и ::after и свойством mix-blend-mode: difference.

4. Адаптивная вёрстка (adaptive.css)
Цель: корректное отображение на семи разрешениях экрана — от десктопа до смартфона. ИИ написал медиазапросы для брейкпоинтов 1920, 1366, 1280, 1024, 768, 414 и 360px.

5. Страница магазина (shop.html)
Цель: отобразить товары с кнопками «Купить». ИИ написал CSS-сетку (grid) с двумя футболками в ряд и лимонадом на всю ширину без обрезки изображения (object-fit: unset).

6. Страница 404 (404.html)
Цель: нестандартная страница ошибки. ИИ написал HTML/CSS с тремя декоративными квадратами и анимацией смены цвета через @keyframes, слайдшоу из фотографий и кнопкой возврата.

7. Страница «О нас» (about.html)
Цель: рассказать о концепции выставки. ИИ написал разметку с двухколоночным блоком (текст + слайдшоу), статичным изображением и цитатой, а также JavaScript для автоматической смены слайдов.

8. Исправление путей для публикации на GitHub Pages
Цель: сайт не работал после выгрузки из-за неверных относительных путей к файлам. ИИ автоматически исправил пути во всех HTML и JS файлах (замена ../assets/ на assets/).

Использованная модель:
Claude Sonnet 4.6 — Anthropic
